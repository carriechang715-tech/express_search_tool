🚀 Vercel 部署指南

5分钟让您的 API 上线，全球访问！

⚠️ 重要提示： GitHub Pages 不支持服务端 API，因此我们需要使用 Vercel 部署 API 功能。
1

访问 Vercel 并登录

点击下面的按钮访问 Vercel：

访问 Vercel

在 Vercel 首页，点击 Sign Up（注册）或 Login（登录）

选择 Continue with GitHub 使用 GitHub 账号登录（推荐）

2

导入 GitHub 项目

登录后，点击右上角的 Add New... → Project

或者直接访问：

创建新项目
选择 Import Git Repository
授权 Vercel 访问您的 GitHub（如果是首次使用）
在仓库列表中找到 express_search_tool
点击 Import
3

配置项目

在配置页面，按以下设置：

  • Project Name: 保持默认或自定义（如：express-api）
  • Framework Preset: 选择 Other
  • Root Directory: 保持默认 ./
  • Build Command: 留空
  • Output Directory: 留空
💡 提示： 因为我们是纯静态站点 + Serverless API，不需要构建步骤。
4

开始部署

确认配置无误后，点击 Deploy 按钮

⏱️ 部署时间：约 30-60 秒

您会看到一个进度动画，等待部署完成...

5

获取 API 地址

部署成功后，您会看到 🎉 Congratulations!

Vercel 会分配一个域名，格式类似：

https://express-search-tool-xxx.vercel.app

您的 API 地址就是：

https://你的域名.vercel.app/api/calculate
6

测试 API

复制下面的命令，替换域名后在终端运行：

curl "https://你的域名.vercel.app/api/calculate?from=CN&to=US&weight=2"

或者在浏览器访问您的 API 测试页面：

https://你的域名.vercel.app/api-test.html

如果返回 JSON 数据，说明部署成功！🎉

✅ 部署成功后，您将拥有：

  • 网页版工具：https://你的域名.vercel.app/
  • API 接口：https://你的域名.vercel.app/api/calculate
  • API 测试页：https://你的域名.vercel.app/api-test.html
  • 自动 HTTPS：所有请求都是安全的
  • 全球 CDN：访问速度快
  • 自动部署：每次 git push 自动更新

🆘 遇到问题？

Q: 部署失败怎么办？

A: 检查 vercel.json 文件是否存在，或查看 Vercel 部署日志

Q: API 返回 404？

A: 确保访问路径是 /api/calculate（有 /api/ 前缀）

Q: 如何自动部署？

A: Vercel 已自动配置，每次 git push 到 main 分支都会自动部署
